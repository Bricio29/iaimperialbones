# Imagens do Widget

Adicione aqui as seguintes imagens em formato PNG antes de fazer o upload para a Kommo:

| Arquivo        | Dimensões recomendadas | Uso                          |
|----------------|------------------------|------------------------------|
| `logo.png`     | 60×60 px               | Ícone do widget no Salesbot  |
| `cover.png`    | 400×272 px             | Capa na página de integrações|

As imagens devem estar nesta pasta (`widget/images/`).
Podem ser a logo da Imperial Bonés ou qualquer imagem representativa.
